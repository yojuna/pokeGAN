# pokeGAN
Use WGANs to create (generate) new kinds of pokemon.

## dependence
```
cv2
tensorflow( >=1.0)
scipy
numpy
jupyter notebook
```
Pokemon training data:
`https://github.com/moxiegushi/pokeGAN/`
